## 六、交互行为总结

| 动作 | 效果 |
|------|------|
| 页面加载 | 存档默认折叠 |
| 点击「📦 存档（N项）▾」 | 展开存档，默认周视图 |
| 点击筛选按钮「周/月/年/全部」 | 切换分组粒度，保持展开 |
| 点击分组标题「▸ 本周…」 | 展开该组，显示任务卡片 |
| 点击存档区域外的位置 | 存档保持当前状态（不自动折叠） |
| 任务完成日期的获取 | `completed_at` > `withdraw_requested_at` > `posted_at` |
