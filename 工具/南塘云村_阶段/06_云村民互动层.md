# 阶段六：云村民互动层

> **目标**：远端的人看得见南塘、说得了话、能互动。数字孪生从"管理工具"变成"共享空间"。
> **状态**：📝 待审查
> **依赖**：阶段一至五（没有前五阶段的数据，云村民就没东西可看）

---

## 一、本阶段要解决什么问题

1. **云村民能看到什么** —— 时间线、地图状态、物品变化、人在做什么
2. **云村民能说什么** —— 弹幕（即时感）、物品留言（长期感）、空间留言
3. **云村民能做什么** —— 打赏、点赞、标记关注

一句话验收标准：**云村民打开云村，能看到南塘现在什么样、最近发生了什么、能对感兴趣的东西留言和打赏。**

## 三、弹幕系统

### 3.1 弹幕设计

弹幕 = 轻量、即时、飘过式的留言。给云村民一种"南塘正在发生什么"的在场感。

```
弹幕特点：
  - 简短（不超过50字）
  - 从右往左飘过（像视频弹幕）
  - 飘过几条后消失，点弹幕区可看历史
  - 不绑定具体物品/空间（那是关联留言的事）
  - 所有人都能发（在地成员 + 云村民）
```

### 3.2 弹幕界面

```
页面底部弹幕区：

┌──────────────────────────────────────────────────┐
│ 🎬 云小方: 好想吃那个饺子！  云小明: 番茄好红！ │
│ 大飞辛苦了！  云小方: 今天南塘天气真好呀        │
├──────────────────────────────────────────────────┤
│ [发一条弹幕...___________________] [发送] [历史] │
└──────────────────────────────────────────────────┘

点击 [历史] → 展开最近的弹幕列表（按时间倒序）
```

### 3.3 弹幕发送

```
输入弹幕 → [发送]

  - 字数限制：50字
  - 频率限制：每30秒最多发1条（防止刷屏）
  - 弹幕保留：最新100条在内存中，更多的存nt_social
  - 不提供NT奖励（弹幕是纯社交，和经济系统无关）
```

## 五、打赏系统

### 5.1 打赏逻辑

云村民可以用NT打赏在地成员（表达喜欢/感谢/鼓励）。

```
打赏流程：

  在时间线/物品/记录上看到 [🎁 打赏] →
    点击 → 弹出打赏框：

  ┌────────────────────────────┐
  │  🎁 打赏给 小杨            │
  │                             │
  │  金额: [10] [30] [50] NT   │
  │        或自定义: [__] NT   │
  │                             │
  │  留言（可选）:              │
  │  [____________________]    │
  │  "饺子看起来太好吃了！"    │
  │                             │
  │  我的余额: 150 NT           │
  │                             │
  │  [✓ 确认打赏]  [取消]      │
  └────────────────────────────┘

确认后：
  → 打赏者余额 -= 打赏金额
  → 被打赏者余额 += 打赏金额
  → 记录事件: tip
  → 被打赏者收到通知
```

### 5.2 云村民如何获得NT

```
云村民初始NT余额为0。

云村民获得NT的方式：
  1. 管理员手动转账（从NT总池划拨）
  2. 被打赏（别人打赏给云村民）
  3. 远期：云村民月费制（每月自动获得一定NT用于互动）

建议初期：管理员给活跃的云村民定期发放少量NT（如每月+50NT），
让他们有"弹药"参与互动。
```

## 六、点赞系统

### 6.1 点赞

对任何记录/物品操作事件都可以点赞：

```
👍 点赞：
  - 每个人对同一条记录只能点赞一次（再点取消）
  - 点赞数显示在记录下方
  - 不产生NT流转
  - 被点赞的记录排序权重提升
  - "最多点赞"可以作为云村民发现好内容的入口
```

  │  📍 更新我的位置            │

## 八、NT 计算逻辑

### 8.1 打赏相关

| 事件 | NT变动 | 说明 |
|------|:---:|------|
| 打赏他人 | 打赏者 -N, 被打赏者 +N | 实时流转 |
| 被打赏 | +N | 自动到账 |
| 打赏留言 | 0 | 附带留言不额外收费 |

### 8.2 互动NT汇总

| 互动 | NT | 限制 |
|------|:---:|------|
| 弹幕 | 0 | 每30秒1条 |
| 物品留言 | 0 | 无限制 |
| 空间留言 | 0 | 无限制 |
| 点赞 | 0 | 每条1次 |
| 打赏 | 自由金额 | 余额足够即可 |

## 十二、待确认问题

1. **弹幕和留言的界限**：目前设计是弹幕=即时飘过、留言=绑定物品/空间。这个区分是否清晰？还是只需要留言就够了？
2. **云村民NT来源**：初期是否由管理员手动发放？每人每月发多少？
3. **打赏是否双向**：在地成员能否打赏云村民？（比如感谢云村民的优质留言）
4. **留言通知**：在地成员是否需要收到"有人评论了你的物品"的通知？建议是。
5. **小人拖拽**：拖拽移动位置是阶段六做还是延期到阶段七？建议先做表单选择（简单），拖拽延后。
