### ~~需要补的~~ ✅ 已全部实现

当前 `renderMyProfile()`（~1521）里有编辑按钮，但没有升级入口。需要在冒险者的「我的资料」底部加升级区域：

```html
<!-- 仅非共建者显示 -->
<div style="margin-top:12px;padding-top:12px;border-top:1px solid var(--border)">
  <div style="font-size:.78rem;font-weight:600;margin-bottom:6px">🔓 升级身份</div>
  <input id="upgradeCode" placeholder="输入邀请码" style="...">
  <button onclick="doUpgrade()">确认</button>
</div>
```

```js
function doUpgrade() {
  var code = document.getElementById('upgradeCode').value.trim();
  if (!code) return;
  var users = getUsers();

  // 检查是否匹配世界码 → 升级为冒险者
  var worldCodes = getWorldCodes();
  if (worldCodes.includes(code)) {
    users[currentUser.name].role = 'adventurer';
    users[currentUser.name].world_id = code;
    saveUsers(users);
    currentUser.role = 'adventurer';
    enterWorld(currentUser);
    return;
  }

  // 检查是否匹配 NPC 邀请码 → 升级为 NPC
  var npcCodes = getNPCInviteCodes();
  if (npcCodes.includes(code)) {
    users[currentUser.name].role = 'npc';
    saveUsers(users);
    saveInviteCodes(npcCodes.filter(function(c) { return c !== code; })); // 消耗
    currentUser.role = 'npc';
    alert('✅ 已升级为南塘在地伙伴！');
    enterWorld(currentUser);
    return;
  }

  alert('邀请码无效');
}
```

### 🔴 待修复：冒险者视角下隐藏「加载」和「导出」

当前 toolbar（~512-517）对所有身份可见：

```html
<div style="display:flex;justify-content:flex-end;align-items:center;gap:8px;margin-bottom:6px">
  <button class="btn" onclick="loadData()">📂 加载</button>
  <button class="btn" onclick="exportData()">💾 导出</button>
  <span style="font-size:.68rem;color:var(--muted)" id="statusDot"></span>
</div>
```

**方案**：利用已有的 `body.builder-mode` / `body.adventurer-mode` CSS class 控制可见性。

**方式一（CSS，推荐）**——在 `<style>` 中加：

```css
body.adventurer-mode .toolbar-admin-only { display: none; }
```

然后给 toolbar 加 class：

```html
<div class="toolbar-admin-only" style="display:flex;justify-content:flex-end;...">
```

**方式二（JS）**——在 `enterWorld()` 的 adventurer/npc/visitor 分支里加：

```js
document.getElementById('adminToolbar').style.display = 'none';
```

**推荐方式一**，因为 CSS 自动跟随 `body` class 变化，预览模式切换时不需要额外 JS。
