## 一、身份层级

```
┌─────────────────────────────────────────┐
│         高级管理员（砚仁 / 1500）          │
│         全局权限，看全部，管全部            │
└─────────────────────────────────────────┘
                    │
        ┌───────────┼───────────┐
        ▼           ▼           ▼
   ┌─────────┐ ┌─────────┐ ┌─────────┐
   │ 南塘NPC │ │  冒险者  │ │  共建者  │
   │ 直接注册 │ │ 直接注册 │ │ 不可直接 │
   │         │ │         │ │ 注册     │
   └────┬────┘ └────┬────┘ └────┬────┘
        │           │           │
        │   ┌───────┘           │
        │   │                   │
        ▼   ▼                   │
   通过营地设置 Step3              │
   「打包命名」认领角色             │
        │                       │
        └───────────┬───────────┘
                    ▼
            ┌──────────────┐
            │   游戏策划组   │
            │  = 共建者身份  │
            │  有任务、材料   │
            │  有结算界面     │
            └──────────────┘
```

## 二、三种入口身份

### 👥 南塘NPC
| 属性 | 说明 |
|------|------|
| 怎么进入 | 直接注册（欢迎画面点击「我是南塘NPC」） |
| 初始状态 | 普通成员，没有角色，没有任务 |
| 数据存储 | `data.members` |
| 怎么变成共建者 | 高级管理员在营地设置 Step3「打包命名」为其认领角色 |
| 认领角色后 | 身份变为共建者/游戏策划组成员 |
| 没认领角色时 | 只是成员档案中的一条记录，没有独立工作界面 |

### ⚔️ 冒险者（玩家）
| 属性 | 说明 |
|------|------|
| 怎么进入 | 直接注册（欢迎画面点击「我是冒险者」） |
| 初始状态 | 玩家身份，可以看到攻略地图 |
| 数据存储 | `data.members`（通过 `isStaff2()` 判断为非工作人员） |
| 怎么变成共建者 | 高级管理员在营地设置中将其认领角色 |
| 认领角色后 | 从冒险者转化为共建者，进入游戏策划组 |

### 🏗️ 共建者（游戏策划组）
| 属性 | 说明 |
|------|------|
| 怎么进入 | **不可直接注册**。通过共建者码/账号登录（欢迎画面点击「我是共建者」） |
| 来源 | 由南塘NPC 或 冒险者 转化而来 |
| 前提条件 | 高级管理员已在营地设置 Step3 为其认领了角色 |
| 看到什么 | 自己被分配的任务、自己的材料表、自己的结算信息 |
| 数据存储 | `data.members` + `staff_cards` 中有其认领记录 |

## 三、高级管理员

| 属性 | 说明 |
|------|------|
| 账号 | 砚仁 |

## 四、身份转化流程

```
南塘NPC 注册 ──→ data.members 中有一条记录
                      │
                      │ 管理员在 Step3 为其认领角色
                      ▼
                data.staff_cards 中生成卡片
                身份变为 共建者/游戏策划组
                      │
                      │ 共建者用账号登录
                      ▼
                看到自己的任务、材料表、结算


冒险者 注册 ──→ data.members 中有一条记录（isStaff2 = false）
                      │
                      │ 管理员在 Step3 为其认领角色
                      ▼
                data.staff_cards 中生成卡片
                身份变为 共建者/游戏策划组
```

## 六、每种身份看到的界面

### 高级管理员
- 看到全部：营地设置(5步向导)、南塘NPC、任务大厅、游戏玩家、攻略地图、日历系统、公会资金池、结算界面
- 可以加载/导出数据、管理模板、执行同步

### 共建者（游戏策划组成员）
- 看到简化界面：
  - 自己被分配的任务列表（可切换状态：待开始/进行中/已完成）
  - 自己的材料表（预算 + 实际支出 + 总额对比）
  - 自己的结算信息（NT、RMB、完成率）
  - 日历系统（只读）
- 不能看到：营地设置、其他人的信息、数据导出、同步操作

### 南塘NPC（未认领角色）
- 看到最简界面：
  - 自己的档案信息
  - 等待被认领的提示
  - 日历系统（只读）

### 冒险者（玩家）
- 看到攻略地图
- 看到每日任务（玩家视角）
- 日历系统（只读）
