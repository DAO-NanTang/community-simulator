## 二、数据从哪来

社区窗口里每一条信息，数据来源都是已有字段，不建新表：

| 显示内容 | 数据来源 | 计算方式 |
|---------|---------|---------|
| 🔥 营地脉动 | `data.tasks` 活跃任务数 | >10 → "热火朝天" / 5-10 → "稳步推进" / <5 → "刚刚开始" |
| 统计数字 | `data.tasks` + `data.tips` | 按 status 计数 / 按 type='gift' 计数 |
| 👤 正在现场 | `data.staff_cards` + `data.tasks` | 共建者从 staff_cards 取；冒险者从 tasks 中有活跃 claim 的人取；每人带他们正在做的任务 |
| 📋 正在招募 | `data.tasks` | 筛选 status='active' + 名额未满 + 无人认领的任务，最多显示 5 个 |
| 💬 打赏面板 | `currentUser` + `calcNtTotal()` | 你的名字 + 你的余额 |
| 🔥 刚刚发生 | `data.tasks` 的 claimants + `data.tips` | 最近提交/完成/领取 + 最近打赏，按时间倒序混排，最多 10 条 |

**营地脉动**的规则：
```
活跃任务 > 10  →  🔥 热火朝天 · N人正在创作
活跃任务 5-10 →  🌿 稳步推进 · N人正在创作
活跃任务 < 5  →  🌱 刚刚开始 · N人正在创作
```

**"正在招募"** 筛选条件：
- `status === 'active'`
- `max_slots > 0` 且活跃认领人数 < `max_slots`
- 或者无认领限制但 claimants 为空
- 按 `posted_at` 倒序，最多 5 条

**"刚刚发生"** 混排逻辑：
- 从 `data.tasks` 收集：最近 submit / complete / claim 事件
- 从 `data.tips` 收集：最近打赏记录
- 合并 → 按时间倒序 → 取前 10 条
- 每条显示：类型图标（⚡=任务动态 / 💬=打赏）+ 描述 + 相对时间

## 三、怎么让外面的人更有兴趣

每一块的设计意图：

| 区块 | 让访客感受到 |
|------|------------|
| 🔥 营地脉动 | **"这个地方是活的"**—不是静态页面，有人在真实创作 |
| 👤 正在现场 | **"这些是真实的人"**—有名有姓有头像，有技能标签，不是抽象的"用户" |
| 📋 正在招募 | **"你也做得到"**—看到具体的任务名字和描述，降低门槛 |
| 💬 送个鼓励 | **"你可以参与"**—不需要加入就能互动，点一下就能送出善意 |
| 🔥 刚刚发生 | **"此刻正在发生"**—时间戳用"3分钟前"，营造实时感 |
| 💡 底部引导 | **"下一步很简单"**—滚动上去输入邀请码就加入 |

打赏按钮的设计意图：
- 免费鼓励 8 个 → 零门槛参与
- 付费礼物 4 档 → 有 NT 的人多一个表达方式
- 没 NT 的人点礼物会弹提示"余额不足" → 不会困惑
- 每个成员卡片上都有打赏按钮 → **看到谁在做有趣的事，立刻就能表达**

## 四、实施步骤

### 第 1 步：撤销错误改动

| 删什么 | 位置 |
|--------|------|
| 村口第 4 张"社区窗口"卡片 | plaza flex 行 |
| `villageModalCommunity` Modal | village modals 区域 |
| `openCommunityWindow()` | JS |
| `openVillageModal`/`closeVillageModal` 的 community 分支 | JS |

### 第 2 步：改造社区副本 Modal

- `#villageModalActivity` 的 `.vm-inner` 加宽到 `max-width:820px`
- "成为冒险者"按钮旁边加 `[👀 浏览营地 ▾]` 标签
- 标签下面加折叠 div `#activityCommunityPanel`

### 第 3 步：写 `renderCommunityWindowCompact()`

新建函数，比当前的 `renderCommunityWindow()` 更紧凑：

**与完整版的区别**：
- 整体字号缩小 15%（适合在 Modal 内展示）
- 成员卡片精简为一行：头像 + 名字 + 技能标签 + 正在做的任务（最多 2 个）
- 统计卡片缩小、横排
- "正在招募"区新增（完整版没有这个）
- "刚刚发生"区新增（完整版没有这个）
- 打赏面板底部加"你的余额"提示
- 底部加引导语

**与完整版的共同点**：
- 都调 `sendTip` 打赏
- 都调 `timeAgo` 相对时间
- 都读 `data.tasks`、`data.staff_cards`、`data.tips`
- 都支持匿名/实名

### 第 4 步：加"浏览营地"切换逻辑

点击标签：
- 第一次：调 `renderCommunityWindowCompact()` → 填入 `#activityCommunityPanel` → 显示 → 标签变「👀 收起营地 ▴」
- 后续：切换显示/隐藏 → 标签文字跟着变
