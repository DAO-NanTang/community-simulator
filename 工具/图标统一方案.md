# 图标统一方案

> 目标：替换现有 emoji 为风格统一、更好看的 SVG 图标
> 2026-07-07

---

## 〇、推荐方案：Phosphor Icons

**Phosphor Icons**（[phosphoricons.com](https://phosphoricons.com)）

| 特性 | 值 |
|------|-----|
| 数量 | 1,512 个 |
| 格式 | SVG / Webfont |
| 风格 | 统一 16×16 网格设计，6 种粗细（Thin/Light/Regular/Bold/Fill/Duotone） |
| 许可 | MIT（免费商用） |
| 引入方式 | 一行 CDN script，零配置 |
| 使用 | `<i class="ph ph-iconname"></i>` |

**为什么选它**：
1. Duotone（双色调）风格有「游戏 UI」的质感，不会太扁平也不会太花哨
2. 6 种粗细可以让不同层级的元素有视觉区分（tab 用 Bold，按钮用 Regular，提示用 Thin）
3. 一个 CDN 标签搞定，不需要 npm、不需要下载、不增加仓库体积
4. MIT 许可证，不担心版权
5. 每个图标都有明确的语义名称，好找

## 二、引入方式

一行 CDN 放在 `<head>` 里：

```html
<script src="https://unpkg.com/@phosphor-icons/web@2.1.1"></script>
```

然后直接用：

```html
<!-- Regular 重量（默认） -->
<i class="ph ph-sword"></i>

<!-- Fill 重量（更醒目） -->
<i class="ph-fill ph-crown"></i>

<!-- Duotone 重量（双色调，最有游戏感） -->
<i class="ph-duotone ph-planet"></i>

<!-- 改颜色 -->
<i class="ph-fill ph-sword" style="color: #5b8cb8"></i>

<!-- 改大小 -->
<i class="ph ph-sword" style="font-size: 24px"></i>
```

## 三、手机端纯图标 Tab 方案

当前 tab 是 emoji + 文字。手机端只显示图标时，Phosphor 图标比 emoji 更好——因为它们是 SVG 矢量图，在任何分辨率下都清晰，而且可以精确控制颜色。

```css
@media (max-width: 768px) {
  .tab { font-size: 0; }           /* 隐藏文字 */
  .tab i { font-size: 1.2rem; }    /* 显示图标 */
  .tab::after { content: attr(data-label); } /* 长按 tooltip */
}
```

```html
<!-- 桌面端 -->
<button class="tab" data-tab="members" onclick="switchTab('members')">
  <i class="ph ph-identification-card"></i> 成员档案
</button>

<!-- 手机端自动隐藏文字，只显示 <i> 图标 -->
```

每个 tab 的 `data-label` 属性存全名，手机端长按/悬停时弹出 tooltip。

## 五、与现有 emoji 的对比

| 对比维度 | 现 emoji | Phosphor SVG |
|---------|---------|--------------|
| 风格一致性 | 差——每个平台显示不同 | 好——统一 16px 网格 |
| 颜色可控 | 否——系统自动着色 | 是——CSS color 精确控制 |
| 大小可控 | 受限——font-size 不精确 | 精确——font-size 或 width/height |
| 暗色适配 | 差——emoji 在暗背景上可能看不清 | 好——SVG 跟随 color 属性 |
| 平台差异 | 大——Windows/Mac/iOS/Android 显示不同 | 无——SVG 在任何平台一样 |
| 专业感 | 低——emoji 偏可爱/休闲 | 高——统一线条的专业感 |
