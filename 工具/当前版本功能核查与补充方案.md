## 二、头像素材库选择器方案

### 功能

点头像 → 弹出网格（60个头像，复用 CHARACTER_SEEDS）→ 选一个 → 保存 → 关闭

### 实现

- 复用成员档案的 `showAvatarPicker()` 逻辑（已有 `CHARACTER_SEEDS` 数组和 `AVATAR_STYLE` 常量）
- 新建 `showWsAvatarPicker()` 函数 — 弹出居中 Modal
- 新建 `wsSelectAvatar(seed)` 函数 — 保存到 `data.members[name].avatar_seed` + `nt_users[name].avatar_seed` + localStorage
- 工作台左侧头像卡绑定 `onclick="showWsAvatarPicker()"`

### 位置

`dashboard.html` 工作台渲染函数 `renderWorkspace()` 第 2719 行

## 四、浏览器缓存问题及解决

### 问题

浏览器缓存旧版 HTML，导致：
- 新加的 JS 函数不加载 → `Unexpected end of input`
- 卡片点不开 → 旧版没有 `toggleCardExpand`
- 头像功能不生效

### 解决

已加入版本标记 `data-version="20260709-3"`，或在 URL 后加 `?t=3` 强制刷新。
