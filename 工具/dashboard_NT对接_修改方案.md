## 一、P0：必须对接（NT 系统运转的基础）

### 1.1 HUD 三栏数值（替换当前 💎 NT）

**当前**：HUD 只显示 `💎 NT 12,500`

**改为**：

```
┌─ HUD ──────────────────────────────────────────────────┐
│  砚仁 (🧙管理员)    🪙 12,500 NT    💠 2,400 贡献    ⭐ 5,800 经验  │
└──────────────────────────────────────────────────────┘
```

- **NT**：从 `data.finance` 汇总（当前已有）
- **贡献值**：新增 `getContributionValue(name)` 函数，从虚拟账本汇总
- **经验值**：新增 `getExperienceValue(name)` 函数，从虚拟账本汇总

**改动**：`updateWorldHUD()` 函数加两行 + 新增两个汇总函数。约 15 行。

### 1.2 虚拟账本自动记账

**核心逻辑**：当任务完成（NT管理员确认通过）时，自动写入虚拟账本。

**当前 flow**：`reviewTask(idx, claimantName, 'approve')` → 更新 claimant.status + 写 data.finance

**改为**：在上述 flow 之后，追加调用 `recordToLedger()`

```js
function recordToLedger(entry) {
  // entry = { type, description, parties: { name: { nt, cv, xp } } }
  var ledger = getVirtualLedger(); // 本月账本
  ledger.entries.push({ id: 'entry-' + Date.now(), date: today, ...entry });
  saveVirtualLedger(ledger);
  // 即时更新贡献值和经验值
  Object.keys(entry.parties).forEach(function(name) {
    if (name === '公共贡献池') return;
    var delta = entry.parties[name];
    if (delta.cv) addContributionValue(name, delta.cv);
    if (delta.xp) addExperienceValue(name, delta.xp);
  });
}
```

**自动记账的触发点**：

| 事件 | 触发函数 | 账本 type | 记账内容 |
|------|---------|-----------|---------|
| 任务审核通过 | `reviewTask(approve)` | `task` | poster -NT/-CV, claimer +NT/+CV |
| 打扫审核通过 | `reviewChore(approve)` (新增) | `cleaning` | claimer +NT/+CV |
| 饭局劳动完成 | `completeMealLabor()` (新增) | `meal_labor` | 劳动者 +NT/+CV |
| 饭局用餐 | `recordMealDine()` (新增) | `meal_dine` | 用餐者 -NT/-CV |
| 空间预定 | `bookSpace()` (新增) | `booking` | 预定者 -NT/-CV |
| 物品购买 | `recordSale()` (新增) | `sale` | 买家 -NT/-CV, 卖家 +NT/+CV(75%), 公共池 +CV(25%) |

**改动**：新增 `getVirtualLedger/saveVirtualLedger/recordToLedger/addContributionValue/addExperienceValue` 5 个函数 + 修改 6 个触发点。约 80 行。

### 1.3 localStorage 新增 Key

| Key | 内容 | 大小 |
|-----|------|------|
| `nt_virtual_ledger` | 当月虚拟账本 | ~5KB/月 |
| `nt_contribution_values` | 各人贡献值 `{ name: cv }` | ~1KB |
| `nt_experience_values` | 各人经验值 `{ name: xp }` | ~1KB |
| `nt_ledger_archive` | 历史月度账本（归档） | ~50KB/年 |
