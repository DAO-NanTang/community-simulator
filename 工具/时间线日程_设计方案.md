## 〇、当前状态分析

### 现有三个视图

| 函数 | 当前行为 | 行数 |
|------|---------|------|
| `renderTimeline()` | 按 deadline 日期分组，左右交替卡片。**无角色过滤** | L5670-5708 |
| `renderCalendar()` | 月历视图，`data.schedule` 驱动，fallback 到 tasks | L5713-5770+ |
| `renderDailyView(role)` | 日视图：里程碑 + 当日任务卡片 + 头像面板。**有 role 参数但只过滤 assignee** | L4716-4900+ |

### 问题

1. **Timeline 分组方式**：按 deadline 日期分组 → 丢失了任务的时间跨度信息（start→deadline）
2. **角色不可见**：Timeline 显示所有任务，adventurer 看到 builder 的任务
3. **无任务状态可视化**：无法一眼看出"哪些在进行""哪些逾期""哪些可领"
4. **三个视图互不关联**：Timeline / Calendar / DailyView 各自独立渲染

## 二、数据来源

### 数据完全来自 `data.tasks[]`

每条任务有：
```js
{
  name: '工笔画临摹',
  start: '2026-07-05',      // 开始日期
  deadline: '2026-07-15',   // 截止日期
  status: '进行中',          // 待开始/进行中/已完成/已确认
  assignee: '若雨',          // 负责人
  type: '支线',
  points: 15,
  scope: 'camp',
  claimants: [{ name: '若雨', status: 'in_progress', ... }]
}
```

### 时间线数据构造

```js
function buildTimelineBars(role, userName) {
  var tasks = data.tasks || [];
  var bars = [];
  
  tasks.forEach(function(t) {
    // 角色过滤
    if (role === 'adventurer') {
      var myClaim = (t.claimants||[]).find(function(c){return c.name===userName;});
      if (!myClaim && t.assignee !== userName) {
        // 可领取的也显示（虚线）
        if (t.status === 'active' && t.scope !== 'personal') return;
        var isClaimable = !myClaim && t.status === 'active';
        if (!isClaimable) return;
      }
    }
    
    bars.push({
      name: t.name,
      start: t.start || t.deadline,
      end: t.deadline || t.start,
      status: getTaskBarStatus(t, userName),
      assignee: t.assignee,
      points: t.points
    });
  });
  
  return bars;
}

function getTaskBarStatus(t, userName) {
  // 返回 bar 颜色状态
  if (t.status === '已完成' || t.status === '已确认') return 'done';
  var myClaim = (t.claimants||[]).find(function(c){return c.name===userName;});
  if (myClaim) {
    if (myClaim.status === 'submitted') return 'submitted';
    if (myClaim.status === 'in_progress' || myClaim.status === 'triggered') return 'active';
  }
  if (t.deadline && t.deadline < new Date().toISOString().slice(0,10)) return 'overdue';
  return 'open';
}
```
