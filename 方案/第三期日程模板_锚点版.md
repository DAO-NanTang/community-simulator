## 四个锚点（自动匹配）

```
        开营日              合照日            展览日         结营日
          │                  │                 │              │
    ┌─────┴─────┐      ┌────┴────┐      ┌─────┴─────┐  ┌───┴───┐
    │ D1 固定    │      │ 营期中点 │      │ 结营-1天   │  │ 最后一天│
    │ ·报到      │      │ ·合照    │      │ ·布展      │  │ ·结营  │
    │ ·欢迎晚会  │      │ ·娱乐    │      │ ·开幕式    │  │ ·返程  │
    │ ·合作社介绍│      │           │      │ ·拍卖会    │  │ ·撤展  │
    └───────────┘      └──────────┘      │ ·茶话会    │  └───────┘
                                          └───────────┘
```

- 开营日 = D1，内容固定不可移动
- 合照日 = 营期中点（Math.floor(天数/2)+1），内容固定
- 展览日 = 结营-1天，内容固定
- 结营日 = 最后一天，内容固定

## 中间填充逻辑

```
展览筹备期（展览日前2-3天）：
  D(展览-3)：装裱
  D(展览-2)：装框准备
  D(展览-1)：布展实施(全天)

其余中间天：
  从模板逐日复制 → 自动标注「📋 复制的，需确认调整」
  用户可以拖拽移动、修改、删除
```

```

② 点「生成日历」
③ 系统自动：
   ┌────────────────────────────────────────────┐
   │ D1  7/20 ← 开营日锚点：报到/欢迎晚会       │
   │ D2  7/21 ← 模板D2 内容（标注需调整）        │
   │ D3  7/22 ← 模板D3                          │
   │ D4  7/23 ← ...                             │
   │ D5  7/24 ← ...                             │
   │ D6  7/25 ← ...                             │
   │ D7  7/26 ← ...                             │
   │ D8  7/27 ← 🎯合照日锚点：合照/娱乐活动      │
   │ D9  7/28 ← 模板D9                          │
   │ D10 7/29 ← ...                             │
   │ D11 7/30 ← ...                             │
   │ D12 7/31 ← 装裱（展览筹备）                 │
   │ D13 8/1  ← 装框准备（展览筹备）             │
   │ D14 8/2  ← 🎭展览日锚点：布展/开幕式/拍卖会 │
   │ D15 8/3  ← 🏁结营日锚点：结营/返程/撤展     │
   └────────────────────────────────────────────┘
④ 用户可以修改D2-D12的任何格子
⑤ 锚点格子的内容不可移动但可微调文字
```

## 锚点锁定

| 锚点 | 位置 | 锁定行为 |
|------|------|---------|
| 开营日 | D1 | 日期不可移，内容可微调 |
| 合照日 | 营期中点 | 日期跟随天数变化自动调整 |
| 展览筹备 | 展览-3 到 展览-1 | 内容固定，日期跟随展览日 |
| 展览日 | 结营-1 | 内容固定，日期跟随结营日 |
| 结营日 | 最后一天 | 内容固定，日期跟随天数 |

## 模板数据存储

```javascript
// 存入 camp_templates，pool: 'schedule'
{
  "第三期日程模板": {
    pool: "schedule",
    slots: [ /* 11个时段，名称用新版 */ ],
    anchors: {
      day1: [ /* 开营日每时段内容 */ ],
      photo_day_offset: "mid",  // 合照日 = 营期中点
      photo_day: [ /* 合照日每时段内容 */ ],
      exhibition_prep: [ /* 展览前3天每时段内容 */ ],
      exhibition_day_offset: "end-1",
      exhibition_day: [ /* 展览日每时段内容 */ ],
      last_day: [ /* 结营日每时段内容 */ ],
    },
    middle_days: [ /* 第2天到展览前4天，每天每时段内容，作为默认复制源 */ ]
  }
}
```
