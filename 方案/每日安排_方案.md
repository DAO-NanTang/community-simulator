# 步骤① 每日安排 — 方案

## 概述

在步骤①「统筹定框架」中，里程碑节点下方增加「每日安排」板块。从粗略的日期+节点，升级为可具体安排每天内容的日历编辑器。

## 核心交互

### 1. 点击编辑
- 点击任意格子 → 弹出编辑框，输入活动名称
- 可从下拉选择已认领的任务（来自步骤②③）

### 2. 拖拽移动
- 长按/拖拽一个格子的内容 → 移到同列另一天
- 例如：把「国画教学」从7/15拖到7/16

### 3. 复制
- 「复制前一天」→ 把前一天的所有内容复制到当天（早上活动除外）
- 点击列头 → 复制整列到剪贴板 / 粘贴

### 4. 从任务包导入
- 下拉选择人物 → 自动把该人物的任务分配到对应日期
- 分配规则：任务的 `deadline` 决定它在哪一天出现
- 没有 deadline 的任务 → 放在右侧「待安排」区，手动拖入

### 5. 默认模板
- 第一次进入时自动生成 8 个时段框架（行动/吃饭沟通/小组会议）
- 基于营期类型预填部分内容

## 数据模型

扩展 `data.schedule`：

```javascript
data.schedule = {
  dates: ['07/15','07/16','07/17',...],
  weekdays: ['星期一','星期二',...],
  slots: [
    { section:'晨修', time:'6:30-7:30', activities:['站桩','行禅','抄经',...] },
    { section:'早餐', time:'7:30-8:00', activities:['','','',...] },
    { section:'上午课程', time:'8:00-11:30', activities:['国画教学','绘画指导',...] },
    // ...
  ]
};
```

每个 `activities[i]` 对应 `dates[i]` 那天的内容。

## 与任务池的关联

步骤②③确定任务和认领人后，步骤①的每日安排可以：

1. 点击格子 → 下拉菜单出现「从任务池选择」
2. 选择任务 → 自动填入任务名 + 编号（如 `E1 国画教学`）
3. 关联关系存在 `data.schedule.task_links[{date,slot,task_name}]`

这样步骤④填写的截止日期会自动反映到每日安排中。

## 实现优先级

| 优先级 | 功能 | 复杂度 |
|--------|------|--------|
| P0 | 点击格子编辑内容 | 低 |
| P0 | 默认时段模板 | 低 |
| P1 | 从任务池下拉选择填充 | 中 |
| P1 | 复制前一天 | 低 |
| P2 | 拖拽移动 | 高 |
| P2 | 任务 deadline 自动分配到日期 | 中 |
