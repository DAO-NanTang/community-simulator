## 一、功能概述

步骤①「统筹定框架」中的每日安排模块。用户设置开营日 + 天数 + 选模板 → 一键生成完整日历。日历支持整列选中、复制插入、拖动。锚点日（开营/合照/展览/结营）自动匹配。确认天数后生成里程碑。

### 列操作

**选中**：点击列内容区（不含日期头行）→ 整列高亮。日期头不可选中。

**插入**：选中列后出现 `[在此列前插入] [在此列后插入]`。插入 = 复制该列内容到新位置，日期自动顺延。`data.camp_dates.duration_days` 自动 +1。

**拖动**：拖住选中列内容 → 拖到目标列 → 两列内容互换。日期头不动。

**删除**：选中列后出现 `[删除此列]`。天数 -1。

### 增加天数后

```
📐 初始设置：7/20 开营 · 15 天
📐 当前实际：7/20 开营 · 17 天（+2天）
[🔮 确认天数并生成里程碑 →]
```

确认后 → 里程碑重算 → 锚点重新匹配。

### 存为模板

点击 `[💾 存为模板]` → 弹出命名框 + 复选框 `☑ 同时保存每日活动内容（作为默认模板）`
- 不勾选 → 只存时段结构（与现有行为一致）
- 勾选 → 同时存锚点内容 + 中间天活动 → 下期可直接填充

## 七、生成逻辑

```javascript
function generateCalendar(startDate, duration, tplName) {
  // 1. 备份
  if (data.schedule && data.schedule.slots && data.schedule.slots.length > 0) {
    data._schedule_backup = JSON.parse(JSON.stringify(data.schedule));
  }

  // 2. 加载模板
  var templates = loadTemplates();
  var tpl = templates[tplName];
  if (!tpl || !tpl.data) { alert('模板无效'); return; }

  // 3. 生成日期列表 → data.schedule.dates / weekdays
  rebuildScheduleDates();

  // 4. 创建 11 个 slot
  data.schedule.slots = (tpl.data.slots || DEFAULT_SLOTS).map(function(s) {
    return { section: s.section, time: s.time, activities: new Array(duration).fill('') };
  });

  // 5. 确定锚点位置
  var day1 = 0;
  var photoDay = Math.floor(duration / 2);
  var exhibitionDay = duration - 2;
  var lastDay = duration - 1;

  // 6. 填入锚点内容 → data.schedule.slots[s].activities[dayIndex]
  if (tpl.data.anchors) {
    fillAnchorActivities(tpl.data.anchors.day1, 0);                              // D1
    fillAnchorActivities(tpl.data.anchors.photo_day, photoDay);                  // 合照日
    fillAnchorActivities(tpl.data.anchors.exhibition_day, exhibitionDay);        // 展览日
    fillAnchorActivities(tpl.data.anchors.last_day, lastDay);                    // 结营日
    // 展览筹备
    (tpl.data.anchors.exhibition_prep||[]).forEach(function(p) {
      var dayIdx = duration - 1 + parseInt(p.offset.replace('end',''));
      var slotIdx = findSlotIndex(p.slot);
      if (slotIdx >= 0) data.schedule.slots[slotIdx].activities[dayIdx] = '🔒 ' + p.activity;
    });
  }

  // 7. 非锚点天从 middle_days 按顺序填充
  var middleIdx = 0;
  var anchorSet = {}; [0, photoDay, exhibitionDay-3, exhibitionDay-2, exhibitionDay-1, exhibitionDay, lastDay].forEach(function(d){ if(d>=0) anchorSet[d]=true; });
  for (var d = 1; d < duration; d++) {
    if (anchorSet[d]) continue;
    var dayData = (tpl.data.middle_days||[])[middleIdx];
    if (dayData) {
      dayData.activities.forEach(function(act, si) {
        if (si < data.schedule.slots.length) data.schedule.slots[si].activities[d] = '📋 ' + act;
      });
    }
    middleIdx++;
  }

  // 8. 标记锚点（存到 camp_dates 供里程碑使用）
  if (!data.camp_dates) data.camp_dates = {};
  data.camp_dates.milestones_anchor_days = { day1:0, photo:photoDay, exhibition:exhibitionDay, last:lastDay };

  localStorage.setItem('camp_data', JSON.stringify(data));
  renderDailySchedule();
}

function fillAnchorActivities(anchor, dayIdx) {
  if (!anchor || !anchor.activities) return;
  anchor.activities.forEach(function(act, si) {
    if (si < data.schedule.slots.length) data.schedule.slots[si].activities[dayIdx] = '🔒 ' + act;
  });
}

function findSlotIndex(slotName) {
  for (var i = 0; i < data.schedule.slots.length; i++) {
    if (data.schedule.slots[i].section === slotName) return i;
  }
  return -1;
}
```

## 八、关键约束

1. 改日期/天数不自动生成日历，必须点按钮
2. 生成前已有数据 → 弹确认 + 自动备份到 `data._schedule_backup`
3. 直接操作 `data.schedule`，步骤⑤渲染 `renderCalendar()` 时读取
4. 日期行不可选中不可拖，仅内容列可操作（选中/插入/拖动/删除）
5. 锚点日内容不可整列拖动
6. 增加天数 > 模板天数 → 超出部分空白
7. 保存模板时复选框决定是否保存活动内容
