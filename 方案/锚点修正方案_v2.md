## 二、报到日（D0）

日历表的第一列 = 报到日（开营前一日），显示 D0。
模板 `activities[0]` 就是报到日的内容：
- 报到、早餐、报到、—、自由活动、午餐&午休、报到、—、自由活动、晚餐、欢迎晚会

## 四、generateMilestones() 修正

```javascript
function generateMilestones() {
  // 从日历实际数据读取（而非输入框）
  var schedDates = data.schedule.dates;
  if (!schedDates || schedDates.length === 0) { alert('请先生成日历'); return; }
  
  // 解析起止日期
  var startMD = schedDates[0];  // "07/20"
  var endMD = schedDates[schedDates.length - 1];
  var parts = startMD.split('/');
  var startD = new Date(2026, parseInt(parts[0])-1, parseInt(parts[1]));
  parts = endMD.split('/');
  var endD = new Date(2026, parseInt(parts[0])-1, parseInt(parts[1]));
  var duration = schedDates.length;
  
  // D0 = 报到日 = startD - 1
  var d0 = new Date(startD); d0.setDate(d0.getDate() - 1);
  
  data.camp_dates.start = startD.toISOString().slice(0,10);
  data.camp_dates.end = endD.toISOString().slice(0,10);
  data.camp_dates.duration_days = duration;
  
  // 18个节点从日历日期推算...
}
```
