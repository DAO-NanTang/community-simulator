# 世界终端 · 原型搭建计划

> 目标：新建一个独立静态网页 `world_terminal.html`，把「世界终端完整愿景」的框架先做出来。不替换现有 dashboard.html，两个文件并存。功能预留、UI 先行。

---

## 一、原型范围：做什么、不做什么

### 做

- 登录/注册页面（头像选择 + 名字 + 密码，纯 UI）
- 共建者：世界终端（5 模式切换 + 世界状态条 + 标签骨架）
- 冒险者：副本游戏（欢迎页 + 我的状态 + 今日委托 + 排行榜）
- 实景地图标签页（Canvas 2D 瓦片地图，素社布局静态展示）
- 两个界面的切换逻辑（共建者 ↔ 冒险者 ↔ 登录页）
- 深色终端风格 + 暖色冒险者风格

### 不做

- 真正的后端认证
- 真实数据 CRUD（用硬编码示例数据展示）
- 5 步创世流程的完整交互（只放按钮和标题）
- 完整表单功能（输入框可显示，但提交只弹 toast "功能预留"）
- 日历甘特图
- 结算计算

## 三、页面结构

### 3.1 三层面板

```
world_terminal.html
    │
    ├── 层 0：登录页（login-screen）
    │     默认显示，登录后隐藏
    │
    ├── 层 1：共建者终端（builder-terminal）
    │     共建者登录后显示
    │
    └── 层 2：冒险者界面（adventurer-view）
          冒险者登录后显示，或共建者切换视角时显示
```

三个层是三个 div，通过 `display: none/block` 切换，无加载时间。

## 五、实现步骤与预估时间

### 第 1 步：HTML 骨架 + CSS 变量（1.5h）

- 创建文件，定义深色终端 + 暖色冒险者两套 CSS 变量
- 三层面板 div 结构
- 响应式基础（手机 375px → 桌面 1200px 都能看）

### 第 2 步：登录页（1h）

- 头像网格选择器（复用 DiceBear API）
- 名字/密码/身份选择表单
- 已有账号头像列表
- 点击进入 → 切换到对应层面板

### 第 3 步：共建者终端框架（2h）

- 世界状态条
- 5 模式切换按钮
- 标签栏（8 个标签，点击切换内容区）
- 创世模式：5 步按钮 + 展开骨架
- 监察模式：全境监控面板骨架

### 第 4 步：冒险者界面框架（1.5h）

- 状态条
- 4 标签切换
- 我的委托：示例任务卡片
- 排行榜：三栏示例数据

### 第 5 步：实景地图 Canvas（2h）

- Canvas 瓦片渲染（学习 Sprite Garden 的 `render` 函数）
- 南塘营地布局硬编码
- 点击检测（像素 → 瓦片坐标）
- 详情弹出面板

### 第 6 步：模式切换 + 数据连接（1h）

- 共建者 ↔ 冒险者 ↔ 登录页的切换逻辑
- 示例数据对象 SAMPLE 挂载到 `window.SAMPLE`
- 所有面板读取 SAMPLE 渲染
- toast 占位函数

### 第 7 步：微调 + 移动端测试（1h）

- 手机浏览器打开测试
- 触摸点击响应
- Canvas 缩放适配
- 细节调整

**总预估：10 小时**
