## 选中一列 + 操作

点击列头 D3 → 整列高亮：

```
  ┌─── 选中 ───┐
  │ 7/22       │
  │  D3        │
  ├────────────┤
  │ 南塘三晨    │
  │ 早餐       │
  │ 走读南塘    │
  │ 社区走访    │
  │ 自由       │
  │ 午休       │
  │ 集体会议    │
  │ 分组建立    │
  │ 自由       │
  │ 晚餐       │
  │  —         │
  └────────────┘

  [在此列前插入一天] [在此列后插入一天]
  插入 = 复制这一列的全部内容到新列
```

## 拖拽整列

拖住 D3 列头 → 拖到 D5 和 D6 之间 → D3 整列移到 D6 前面，原 D3 位置变为空。

## 增加了2天

用户加了2天 → 延期内容自动出现：

```
│ 时段               │...│ 8/2 │ 8/3 │ 8/4  │ 8/5  │
│                     │   │ D14 │ D15 │ D16  │ D17  │
│ ────────────────────────────────────────────────── │
│ 🌅 晨起·南塘三晨    │   │南塘三晨│结营│ 村展  │补充装裱│
│ ...                 │   │      │    │      │      │
│ 🌙 晚间活动         │   │茶话会 │结束│大地书房│快闪展│

📐 初始设置：7/20 开营 · 15 天
📐 当前实际：7/20 开营 · 17 天（+2天）
[🔮 确认天数并生成里程碑 →]
```

## 点击确认后 → 里程碑更新

```
📐 实际天数确认为 17 天
    │
    ▼
里程碑重算：开营日 7/20，17天 → 结营日 8/5
    │
    ├── 展览日 = 8/4（结营-1）
    ├── 合照日 = 7/28（营期中点）
    └── 18个节点日期全部刷新
    │
    ▼
锚点内容自动重新匹配到正确日期
```
