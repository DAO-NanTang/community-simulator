````javascript
function generateCalendar(startDate, duration, tplName) {
  1. 备份 data.schedule → data._schedule_backup
  2. 从 camp_templates 加载模板（pool='schedule'）
  3. 计算锚点位置（offset 体系见模板 JSON 注释）：
     // end     = 结营日 = duration - 1
     // exhibition = 展览日 = duration - 2 (= end-1)
     // exhibition-1 = 展览前1天 = duration - 3 (= end-2)
     // exhibition-2 = 展览前2天 = duration - 4 (= end-3)
     // exhibition-3 = 展览前3天 = duration - 5 (= end-4)
     // mid     = 合照日 = Math.floor(duration / 2)
     day1 = 0
     photoDay = Math.floor(duration / 2)
     exhibitionDay = duration - 2
     lastDay = duration - 1
  4. 创建 11 个 slot，每个 activities = new Array(duration).fill('')
  5. 填入锚点内容 → 写入 data.schedule.slots[s].activities[dayIndex]
     · anchors.day1 → day1
     · anchors.photo_day → photoDay
     · anchors.exhibition_prep（按 offset 匹配 exhibition-3/-2/-1）
     · anchors.exhibition_day → exhibitionDay
     · anchors.last_day → lastDay
  6. 非锚点天从模板 middle_days 依次复制 → activities[dayIndex] = 模板值，标注「📋」
  7. 锚点天在 data.camp_dates.milestones 中标记 is_anchor: true
  8. 存入 data.schedule，渲染表格
}
````
